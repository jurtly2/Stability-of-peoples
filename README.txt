# The Stability of Peoples — Guided Reader V1

This is a rebuilt version focused on easy navigation and mobile readability.

## Open
Open `index.html`.

## Main pages
- `index.html` — Start
- `reader.html` — guided passage reader
- `doctrine.html` — doctrine lines saved so far
- `glossary.html` — plain definitions
- `print.html` — doctrine-only PDF export page

## Reader modes
The reader supports:
- Exact Smith
- Plain Smith
- Compare
- Our Answer

Example URLs:
- `reader.html?id=intro-01&view=compare`
- `reader.html?id=intro-03&view=answer`

## Editing content
Edit `assets/content.js`.

## Mobile
The site uses a fixed bottom nav on mobile:
Start | Reader | Doctrine | Glossary | Print
